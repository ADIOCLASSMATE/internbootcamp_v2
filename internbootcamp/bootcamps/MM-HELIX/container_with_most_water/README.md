# 基于 Container With Most Water 任务的大语言模型多模态推理能力评测

## 1. 任务描述
Container With Most Water 任务要求模型结合图像信息（柱状图）与文本指令，找出能容纳最多水的两条垂线。该任务考察了模型的视觉感知能力（Visual Perception）和逻辑推理能力（Logical Reasoning）。

## 2. 评测设置

### Prompt Optimization
我们设计了优化的 Prompt，引导模型仔细分析图像，并按 `\boxed{integer}` 格式输出最大容积。

### Reward Function
采用严格的 **Binary Reward** 机制：
- **Answer Reward**: 预测答案与真实答案完全一致得 **0.9 分**，否则 **0.0 分**。
- **Format Reward**: 格式正确得 **0.1 分**。

## 4. 结果分析与论文对比

### 难度分析
随着难度等级提升（数据范围扩大、列表长度增加），模型性能普遍下降。然而，大参数模型（如 Qwen3-VL-235B）在 Level 5 等高难度设置下依然保持了 50.50% 的准确率，展现了强大的鲁棒性。

### 与论文基准对比
实验结果表明，我们的评测数据与论文基准均呈现出显著的 **Scaling Law** 趋势。下表展示了 Qwen2.5-VL 系列模型在我们的评测中与论文基准的得分对比：

| Model | Paper Score (%) | Our Score (%) |
| :--- | :--- | :--- |
| Qwen2.5-VL-7B-Instruct | 6.7 | 11.6 |
| Qwen2.5-VL-32B-Instruct | 16.7 | 21.7 |
| Qwen2.5-VL-72B-Instruct | 26.7 | 36.1 |

- **性能提升**: 我们的评测结果在各尺寸模型上均**优于论文基准**。这主要归功于我们优化的 Prompt Template，它更有效地引导模型进行图像分析和逻辑运算。

### Scaling Law
从 Qwen2.5-VL 到 Qwen3-VL，随着参数量的增加，模型性能实现了飞跃。特别是 Qwen3-VL-235B，以 71.20 的高分领跑，证明了大规模多模态模型在解决算法类视觉问题上的巨大潜力。

## 5. 结论
本次评测验证了 Qwen-VL 系列模型在多模态推理任务上的能力。优化的 Prompt 能显著提升表现，而模型规模（Scaling）依然是突破复杂推理瓶颈的关键。
